# spotter-eld-planner

A full-stack ELD trip planning app built with Django and React.

## Stack
- **Backend:** Django, Django REST Framework
- **Frontend:** React, Vite, Tailwind CSS, React-Leaflet
- **Mapping:** OpenStreetMap + OSRM (free, no API key)

## Features
- Route planning with geocoding
- FMCSA HOS-compliant schedule generation (70hr/8-day rule)
- ELD Daily Log sheet generation
- Interactive route map
